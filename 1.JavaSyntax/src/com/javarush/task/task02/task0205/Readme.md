<h3>Прибавка к зарплате</h3>
<p>Реализуй метод <mark>public static void salary(int <strong>a</strong>)</mark>.<br />
Метод должен увеличить переданное число на <strong><span class="text-user">100</span></strong> и вывести на экран надпись: «<em><strong>Твоя зарплата составляет: </strong></em><span class="text-user"><strong>a</strong></span><em><strong> долларов в месяц.</strong></em>»<br />
Где <mark class="user">a</mark> &#8212; это переданное число, которое увеличили на <strong><span class="text-user">100</span></strong>.</p>
<p><strong>Пример вывода на экран для</strong> <mark class="user">а</mark> = <strong><span class="text-user">700</span></strong>:<br />
<code><span class="text-green">Твоя зарплата составляет: 800 долларов в месяц</span></code></p>
